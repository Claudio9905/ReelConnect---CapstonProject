# ReelConnect - CapstonProject
PRESENTAZIONE DEL PROGETTO:
ReelConnect, "Reel" come bobina cinematografica + "Connect" come connessione sociale.
Il progetto si baserà su un'applicazione web full-stack dedicata al mondo cinema.

Gli utenti avranno la propria pagina profilo dove potersi presentare, scegliere i film e le serie TV viste o da vedere, pubblicare post ( in gergo "CIAK") che potranno essere visti dai propri iscritti (in questo caso "FAN"). Per ogni film o serie TV si potrà scegliere l'attore/attrice preferito/a e dare una valutazione alla pellicola.
Ci sarà una sezione home, chiamata "CineVerse", un multiverso di CIAK condivisi dagli utenti;
una chat, "BehindTheChat" (invece di "Behind the scenes"), per interagire con altri utenti con la possibilità di creare dei gruppi.
Per un tocco di sana competizione, non mancherà il "Ciak! Si risponde!". un quiz a tema dove ogni utente potrà mettere alla prova la propria conoscenza cinefila, scalando la classifica di altri appassionati.

OBIETTIVO DEL PROGETTO:
Una piattaforma social dedicata alla condivisione dei propri pensieri, critiche e valutazioni sui contenuti cinematografici (tra film e serie TV). 

TECNOLOGIE USATE:

- LATO FULL-STACK:

 HTML5,CSS3,JAVASCRIPT;
 BOOTSTRAP, SASS;
 NPM, VITE;
 REACT + REACT-BOOTSTRAP + REACT-ROUTER + REACT-REDUX + REACT-TYPESCRIPT;	

- LATO BACK-END:

 JAVA;
 SPRING, SPRINGBOOT (SPRING DATA JPA + SPRING WEB + SPRING SECURITY);
 POSTGRESQL (Lato Database);  
 Postman (per l'interazione con le API REST)
